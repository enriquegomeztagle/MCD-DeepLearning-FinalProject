# MCD Deep Learning - Final Project

React + FastAPI application deployed on AWS ECS Fargate via CDK.

## Architecture

```
Frontend (React + Nginx) → Backend (FastAPI) → ML Model
         ALB                      ALB
         ECS Fargate              ECS Fargate
```

Infrastructure: VPC, ECS Cluster, 2x Fargate Services, 2x ALBs, Secrets Manager (API Key), IAM Roles, CloudWatch Logs.

## Prerequisites

- Docker Desktop
- Node.js (v22+)
- Python 3.12+
- AWS CLI

## Local Testing (no AWS account needed)

One command deploys the full AWS infrastructure locally using [LocalStack](https://localstack.cloud):

### Setup

1. Get a free LocalStack auth token at https://app.localstack.cloud (Student/Hobby plan)
2. Edit `.env` and replace the token with yours:
   ```
   LOCALSTACK_AUTH_TOKEN=your-token-here
   ```

### Deploy

```powershell
.\deploy-localstack.ps1
```

This will:
1. Build Docker images (backend + frontend)
2. Start LocalStack (AWS emulator)
3. Synthesize CDK CloudFormation template
4. Deploy 50 AWS resources (VPC, ECS, ALB, Secrets Manager, IAM...)
5. Run ECS containers with your app
6. Print URLs to access the app

Output:
```
═══════════════════════════════════════════
  ✅ Deployment complete!
═══════════════════════════════════════════

  🌐 Frontend:    http://localhost:<port>
  🔧 Backend API: http://localhost:<port>/docs
  🔑 API Key:     <auto-generated>
```

### Inspect Resources

Open https://app.localstack.cloud/ to see all deployed AWS resources (ECS, VPC, ALB, Secrets, etc.)

### Tear Down

```powershell
.\deploy-localstack.ps1 -Down
```

## Deploy to AWS (production)

```bash
cd infrastructure
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
cdk bootstrap
cdk deploy
```

CDK automatically builds Docker images, pushes to ECR, and deploys on ECS Fargate.

## API

| Endpoint | Method | Auth | Description |
|----------|--------|------|-------------|
| `/health` | GET | No | Health check |
| `/v1/models` | GET | No | List available models |
| `/v1/predictions` | POST | `X-API-Key` header | Run inference |

### Example

```bash
curl -X POST http://localhost:<port>/v1/predictions \
  -H "Content-Type: application/json" \
  -H "X-API-Key: <api-key>" \
  -d '{"instances": [{"input": "test"}]}'
```
